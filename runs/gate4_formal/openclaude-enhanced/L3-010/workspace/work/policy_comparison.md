# 审批线版本对比

## 旧版 (2022版, 已废止)
- 文号: XX证财规〔2022〕乙号
- 效力: 已废止, 不得作为现行审批依据
- 审批线:
  - ≥8000元 → 部门总经理审批
  - ≥30000元 → 分管副总审批

## 新版 (2025修订版, 现行有效)
- 文号: XX证财规〔2025〕甲号
- 效力: 现行有效, 自2025-01-01起施行
- 审批线 (按《授权管理办法》附件二):
  - AR-01: [0, 3000) → 部门经理
  - AR-02: [3000, 10000) → 部门经理 + 财务复核
  - AR-03: [10000, 50000) → 部门总经理
  - AR-04: [50000, 200000) → 分管副总
  - AR-05: ≥200000 → 总经理办公会

## 版本陷阱区间
- 8000-9999: 旧=部门总经理, 新=部门经理+财务复核
- 30000-49999: 旧=分管副总, 新=部门总经理
