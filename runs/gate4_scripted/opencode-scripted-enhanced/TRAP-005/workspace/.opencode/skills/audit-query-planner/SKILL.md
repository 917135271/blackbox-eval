# Audit Query Planner

Use only when `work/scripted_workflow.json` lists this Skill.

Identify the requested rule, population, period, required fields, aggregation unit, exceptions, and stopping condition. Keep the plan in the active reasoning context; workflow state and checkpoint files are maintained by scripts.

Do not broaden a single-rule question. Do not create findings during planning. Delegate only an independent, bounded investigation whose expected benefit justifies the extra time.
