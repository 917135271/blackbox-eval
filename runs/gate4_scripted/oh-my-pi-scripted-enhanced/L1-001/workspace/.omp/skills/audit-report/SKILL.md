# Audit Report

Separate verified facts, applicable rules, judgment, impact, and recommendations. Reconcile all IDs, dates, amounts, and counts with retrieved evidence. State both the threshold and observed value for every boundary judgment.

Keep unrelated findings and calculation-only records out of the result. Use the explicit phrase `无异常` when no anomaly exists. The scripted control layer handles JSON artifacts, evidence matrix construction, validation, memory, and submission preparation.
